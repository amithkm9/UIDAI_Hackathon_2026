Methodology

This section details the systematic approach adopted for data cleaning, preprocessing, and transformations applied before analysis.

3.1 Data Cleaning & Preprocessing

Step Operation Rationale
1. Data Loading Concatenate multiple CSV chunks per dataset Handle large files split by UIDAI API
2. Date Parsing Convert dd-mm-yyyy string to datetime Enable temporal analysis and time-series operations
3. Text Standardization Strip whitespace, apply Title Case to state/district Ensure consistent grouping and joins
4. Missing Values Check for nulls, fill with 0 for numeric columns Prevent calculation errors in aggregations
5. Feature Engineering Create derived columns (total_enrol, total_demo, total_bio) Simplify downstream analysis

3.2 Data Transformation Pipeline

# Step 1: Load and merge datasets from chunked CSV files
df_bio = pd.concat([pd.read_csv(f) for f in bio_files], ignore_index=True)
df_demo = pd.concat([pd.read_csv(f) for f in demo_files], ignore_index=True)
df_enrol = pd.concat([pd.read_csv(f) for f in enrol_files], ignore_index=True)

# Step 2: Data Cleaning & Standardization
for df in [df_bio, df_demo, df_enrol]:
    df['date'] = pd.to_datetime(df['date'], format='%d-%m-%Y')
    df['state'] = df['state'].str.strip().str.title()
    df['district'] = df['district'].str.strip().str.title()

# Step 3: Feature Engineering - Create aggregated totals
df_bio['total_bio'] = df_bio['bio_age_5_17'] + df_bio['bio_age_17_']
df_demo['total_demo'] = df_demo['demo_age_5_17'] + df_demo['demo_age_17_']
df_enrol['total_enrol'] = df_enrol['age_0_5'] + df_enrol['age_5_17'] + df_enrol['age_18_greater']

# Step 4: Save to optimized Parquet format for faster subsequent analysis
df_enrol.to_parquet('processed_data/enrolment_clean.parquet', index=False)

Key Algorithms Implemented

Migration Flow Detection

# Innovation 1: Migration Detection from Demographic Updates
state_enrol = df_enrol.groupby('state')['total_enrol'].sum()
state_demo = df_demo.groupby('state')['total_demo'].sum()

# Migration Index = Demo Updates / New Enrollments
migration_df['migration_index'] = (total_demo / total_enrol).round(2)

# Classification Logic
median_index = migration_df['migration_index'].median()
migration_df['migration_type'] = migration_df['migration_index'].apply(
    lambda x: 'Migration Hub' if x > median_index * 1.5 
    else ('Migration Source' if x < median_index * 0.7 else 'Balanced')
)

Service Desert Detection Algorithm

# Innovation 4: Identify Underserved Areas
district_stats = df_enrol.groupby(['state', 'district']).agg({
    'pincode': 'nunique',
    'total_enrol': 'sum'
}).reset_index()

# Service Density Calculation
district_stats['enrol_per_pincode'] = total_enrol / unique_pincodes

# Desert Score (0-100, higher = more underserved)
district_stats['desert_score'] = (
    100 - district_stats['enrol_per_pincode'].rank(pct=True) * 100
).round(0)

# Priority Classification
district_stats['priority'] = district_stats['desert_score'].apply(
    lambda x: 'Critical' if x >= 80 else ('High' if x >= 60 else 'Medium')
)

SDG Alignment Score Calculation

# Innovation 5: Link to UN Sustainable Development Goals
# SDG Component Scores
sdg_df['sdg_16_9_identity'] = (total_enrol / population) * 100  # Legal Identity
sdg_df['sdg_1_3_protection'] = (adult_enrol / adult_population) * 100  # Social Protection
sdg_df['sdg_4_1_education'] = (child_enrol / child_population) * 100  # Education
sdg_df['sdg_10_2_inclusion'] = (pincodes / max_pincodes) * 100  # Geographic Spread

# Weighted Overall Score (40% + 25% + 20% + 15% = 100%)
sdg_df['sdg_alignment_score'] = (
    sdg_16_9.rank(pct=True) * 40 + sdg_1_3.rank(pct=True) * 25 +
    sdg_4_1.rank(pct=True) * 20 + sdg_10_2.rank(pct=True) * 15
)

Age Cohort Forecasting

# Innovation 3: Predict Future Service Demand
# Current infants (0-5) will need biometric update at age 15 (in 2031)
state_age['bio_demand_2031'] = state_age['age_0_5']

# Current children (5-17) will need adult services in 5 years
state_age['demo_demand_2031'] = state_age['age_5_17']

# Growth Potential Score
state_age['growth_potential'] = ((age_0_5 + age_5_17) / total_enrol * 100)

Discovery: 268 districts identified as "service deserts" with critically low enrollment-per-pincode ratios - affecting 90 million citizens.

Methodology

Service Density = Total Enrollments / Unique Pincodes
Desert Score = 100 - Percentile Rank of Service Density
Service Desert = Desert Score > 70 (bottom 30%)
